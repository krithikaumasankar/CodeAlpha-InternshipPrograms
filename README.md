# CodeAlpha Internship Programs

This repository contains the Python projects I completed during my **CodeAlpha Python Programming Internship**. Each project focuses on improving problem-solving skills and applying Python concepts such as file handling, regular expressions, object-oriented programming, and basic application development.

---

## 📂 Projects

### 🤖 AI Chatbot
A simple rule-based chatbot that interacts with users by responding to predefined questions and greetings.

**Concepts Used**
- Conditional statements
- Loops
- String handling
- Functions

---

### 📧 Email Extraction
A Python program that extracts valid email addresses from a text file using **Regular Expressions (Regex)**.

**Files**
- `email_extractor.py`
- `input.txt`
- `emails.txt`

**Concepts Used**
- File Handling
- Regular Expressions (`re`)
- Reading and Writing Files

---

### 🎮 Hangman Game
A console-based implementation of the classic Hangman game where players guess a hidden word one letter at a time.

**Concepts Used**
- Random module
- Loops
- Lists
- Strings
- Functions

---

### 📈 Stock Portfolio Tracker
A simple portfolio management program that allows users to record stock investments and calculate the total portfolio value.

**Files**
- `stock_portfolio.py`
- `portfolio.txt`

**Concepts Used**
- Dictionaries
- File Handling
- User Input
- Basic Calculations

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/krithikaumasankar/CodeAlpha-InternshipPrograms.git
```

Navigate to the repository:

```bash
cd CodeAlpha-InternshipPrograms
```

Run any project:

```bash
python AI_Chatbot/chatbot.py
```

```bash
python Email_extraction/email_extractor.py
```

```bash
python Hangman_game/hangman.py
```

```bash
python Portfolio_maker/stock_portfolio.py
```
